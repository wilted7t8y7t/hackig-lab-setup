🛡️ Ethical Hacking Lab Setup Guide for Buaa (Beginner to Pro)

📌 1. What is Ethical Hacking?

Ethical hacking is legally testing a system's security to find vulnerabilities. It helps organizations fix weaknesses before real hackers exploit them.

💿 2. Install Kali Linux with VirtualBox

sudo apt update && sudo apt install virtualbox -y

Download ISO: kali.org

  • Open VirtualBox → New → Name: Kali Linux
  • Type: Linux, Version: Debian (64-bit)
  • RAM: 2GB+, HDD: 20GB+
  • Attach Kali ISO → Start and install

👤 3. Create New User & Update System

sudo adduser buaa sudo usermod -aG sudo buaa sudo apt update && sudo apt upgrade -y

🧰 4. Important Kali Linux Tools & Uses

  • nmap: Network scanning → nmap -sV target_ip
  • wireshark: Network packet analysis
  • burpsuite: Web app vulnerability testing
  • metasploit: Exploit development
  • john: Password cracking
  • hydra: Brute-force login testing
  • hashcat: Advanced password cracking
  • aircrack-ng: Wi-Fi hacking
  • sqlmap: SQL injection automation
sudo apt install nmap wireshark burpsuite metasploit-framework john hydra hashcat aircrack-ng sqlmap -y

📡 5. Networking & Route Setup

ip a sudo ip route add default via 192.168.1.1 sudo hostnamectl set-hostname buaa-hacker sudo nano /etc/hosts

Add this line:

127.0.1.1 buaa-hacker
